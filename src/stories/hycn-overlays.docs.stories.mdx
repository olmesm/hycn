# Overlays and disclosure

- `hycn-disclosure` exposes button-controlled content with `open`, `disabled`, and `hycn-toggle`.
- `hycn-popover` coordinates a slotted trigger and generic dialog-like popup, including outside and Escape dismissal.
- `hycn-tooltip` associates a delayed `role="tooltip"` description with a slotted trigger on hover and focus.
- `hycn-toast` owns a polite/assertive live-region queue through `show(message, tone)` and `dismiss(id)`.

All components expose their surfaces through CSS parts and preserve focus-visible native controls.
