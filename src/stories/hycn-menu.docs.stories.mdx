# Menu

`hycn-menu` coordinates a trigger and a popup menu, including focus movement, outside-pointer dismissal, and focus restoration.

## Usage

```html
<hycn-menu label="Actions">
  <button slot="trigger" type="button">Actions</button>
  <button data-value="edit" type="button">Edit</button>
  <button data-value="archive" type="button">Archive</button>
</hycn-menu>
```

## API

| Property | Attribute | Default | Purpose |
| --- | --- | --- | --- |
| `open` | `open` | `false` | Controls popup visibility. |
| `label` | `label` | `"Menu"` | Accessible menu name. |

Event: `hycn-select` contains `{ value }`, using `data-value` or item text. Slots: `trigger`, default menu items. Part: `menu`. Theme with `--hycn-menu-background`, `--hycn-menu-border`, `--hycn-menu-radius`, and `--hycn-menu-z-index`.

## Keyboard behavior

Enter, Space, ArrowDown, or ArrowUp opens from the trigger. Arrow keys, Home, and End navigate enabled items. Escape closes and restores trigger focus; Tab closes without interrupting normal focus movement.
