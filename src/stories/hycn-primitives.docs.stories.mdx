# Presentational primitives

HYCN includes `alert`, `avatar`, `badge`, `breadcrumb`, `button`, `card`, `flex`, `icon`, `image`, `list`, `skeleton`, `table`, `tag`, and `text` primitives. Each uses the nearest native semantic element, exposes a `base` or content-specific CSS part, and stays intentionally small for composition.
